labelbuddy example files

The files listed here provide examples of formats that can be used to import documents and labels into a labelbuddy database. The files whose name contains exported have been exported from labelbuddy. The import and export formats are the same, which means that these exported files can also be imported into labelbuddy. All the formats are described in the documentation.

Demo documents and labels

These are copies (in json format) of some of the documents and labels used to fill the demo database when you start labelbuddy with the --demo option or select File  Demo in the GUI. The documents are short extracts from Wikipedia.
